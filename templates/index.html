Bienvenido a la App de los perros

¿Para que puede servir la aplicacion?

DoggApp es una aplicación web desarrollada con Flask que permite consultar información básica de distintas razas de perros. El usuario puede buscar una raza mediante un formulario y la aplicación obtiene los datos desde una API externa, mostrando la información de forma dinámica en pantalla.
