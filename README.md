Breve descripcion:

DoggApp es una aplicación web sencilla desarrollada con Flask que permite buscar información sobre diferentes razas de perros
la aplicación cuenta con varias rutas, plantillas con herencia y un formulario donde se ingresa el nombre de la raza
al enviar el formulario, la aplicación se conecta a una API externa (TheDogAPI) para obtener datos reales como el nombre de la raza, su temperamento, peso, altura e imagen.
La información se muestra de forma dinámica en la página de resultados utilizando plantillas HTML y Bootstrap para el diseño.
